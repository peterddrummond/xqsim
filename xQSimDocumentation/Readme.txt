(1) The phase-space simulation code is xqsim.m, in the XQSIMCODE folder.

(2) This is called with some parameters. Variety of examples are in the XQSIMEXAMPLES folder.

(3) There are test scripts: xqsim_test**, with different parameters.

(4) There are experimental comparison scripts called qsim_experiment** as well.

(5) Experimental data-files are generated by the qcount.m, qcount_le or qcount_be code.

(6) Read the manuals for details: xGRAPH is a compatible graphics program.
